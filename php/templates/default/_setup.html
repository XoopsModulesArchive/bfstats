update from 0.2c
Use this method, if you already have an old data-modell of select(bf) in your database.

Warning
There are new features in the 0.3-release that need the new parser to add specific information to your database. When you update the 0.2c datamodell to 0.3 than you of course keep the old data. BUT you'll have to deal with the following conditions:
  • every kind of data that are new in 0.3 will not be available for your old data (playtimes, etc.)
  • your stats will not show any information, until you parsed some files with the 0.3 parser. This is because the parser has to cache some data which are used in the new PHPs to build up the ranking!
  • The types of lots of fields in the tables have been changed for more speed. These changes will not be applied to your datamodell.
  • you'll not be able to login into your 0.2c Admin-Mode anymore, using you old pages. The password has been encrypted in 0.3 and 0.2c cannot deal with that.
Anyway, the worst that could happen is that select(bf) will not work correctly.
